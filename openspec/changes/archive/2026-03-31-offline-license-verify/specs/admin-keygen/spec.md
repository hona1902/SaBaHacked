## ADDED Requirements

### Requirement: Admin Keygen HTML Tool
The system SHALL provide a standalone HTML file (`keygen.html`) that allows the admin to generate license keys from machine codes. This file SHALL NOT be packaged in the extension.

#### Scenario: Admin generates license key
- **WHEN** admin opens `keygen.html` in any modern browser, enters an 8-character machine code, and clicks "Tạo Key"
- **THEN** the tool SHALL display a 16-character uppercase license key computed as `HMAC-SHA256(machineCode, SECRET_KEY)[0:16].toUpperCase()`

#### Scenario: Invalid machine code input
- **WHEN** admin enters a machine code shorter than 8 characters or containing non-hex characters
- **THEN** the tool SHALL show a validation error and not generate a key

### Requirement: Keygen uses same secret as extension
The keygen tool MUST use the identical SECRET_KEY as `license.js` to ensure generated keys are valid for verification.

#### Scenario: Key generated by keygen validates in extension
- **WHEN** admin generates a key for machine code "ABCD1234" using `keygen.html`
- **AND** user with machine code "ABCD1234" enters that key in extension Settings
- **THEN** the extension SHALL verify the key as valid
